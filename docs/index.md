# jWaves Knowledge Base

Welcome to the official documentation for the jWaves SaaS platform.

Use this space for:

- Product onboarding guides
- Feature walkthroughs
- Admin and billing operations
- API integration references
- Support FAQs and release notes

## Suggested authoring workflow

1. Add a feature page in the most relevant section.
2. Link it in `mkdocs.yml` navigation.
3. Run `mkdocs build --strict` before pushing.
